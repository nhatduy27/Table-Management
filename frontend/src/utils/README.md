# Utils

## Người 3 - Frontend UI

Folder này chứa các utility functions và constants.

### Files cần tạo:
- `constants.js` - App constants
  - LOCATIONS: ['Indoor', 'Outdoor', 'Patio', 'VIP Room']
  - STATUS: ['active', 'inactive']
  - API endpoints
  
- `helpers.js` - Helper functions
  - formatDate(date)
  - downloadFile(blob, filename)
  - showErrorToast(message)
  - showSuccessToast(message)
