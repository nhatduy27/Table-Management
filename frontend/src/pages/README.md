## Người 3 - Frontend UI

Folder này chứa các page components (full pages).

### Files cần tạo:
- `TableManagement.jsx` - Main page for table management

  - Handle CRUD operations
  - Manage state for tables
  
- `TableDetails.jsx` (Optional) - Detailed view of single table
  - Show table info
  - Display QR code
  - Edit/Delete actions
