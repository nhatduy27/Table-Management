# Components

## Người 3 - Frontend UI

Folder này chứa các React components.

### Files cần tạo:

#### Table Management Components:
- `TableList.jsx` - Display tables in grid/table view
- `TableCard.jsx` - Single table card component
- `TableForm.jsx` - Form create/edit table (modal or separate page)
- `TableFilter.jsx` - Filter by status, location
- `ConfirmDialog.jsx` - Confirmation dialog for delete/deactivate

#### QR Code Components:
- `QRCodeDisplay.jsx` - Display QR code with react-qr-code
- `QRCodeModal.jsx` - Modal to show QR code details
- `QRDownloadButtons.jsx` - Download PNG/PDF buttons
- `QRRegenerateButton.jsx` - Regenerate QR with confirmation
